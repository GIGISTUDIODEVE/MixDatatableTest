# 가족 협업형 섬 경영 & 디펜스 게임 (Host A / Client B) - 구조 정리 README

> 목적: 부모(호스트 A)와 자식(클라이언트 B)이 함께 즐기는 **퀘스트 기반 성장형** 섬 경영 + 디펜스 게임의 전체 구조를 한 문서로 정리합니다.  
> 핵심: **A가 퀘스트/보상/지원**, **B가 플레이/성장/합성**.

---

## 1) 한 줄 요약
**A(부모)가 일일 퀘스트와 보상을 설계하고 지원하며, B(자식)가 섬을 운영/방어하고 몬스터를 수집·확률 합성으로 성장**하는 가족 협업 게임.

---

## 2) 역할 및 권한

### 2.1 Host A (부모) — 관리자 + 후원자
- **일일 퀘스트 생성/등록**: 목표와 보상 설정
- **포인트 충전**: 광고 보기 / 현금 구매 / 출석
- **B 섬 방문**: 언제든 접속하여 진행 상태 확인
- **도움 제공**: 팝업 버튼으로 건설/업그레이드 시간 단축 등 *가속 지원*
- (추천) 합성 관련 지원: **안정제/보호권/촉매** 같은 *안전장치성 보상* 제공

### 2.2 Client B (자식) — 플레이어
- **퀘스트 수행/완료**
- **섬 경영**: 농사, 나무심기, 건물 건설/업그레이드
- **전투(디펜스)**: 적 몬스터 공격 방어
- **몬스터 수집/육성/합성**: 알 구매 + 확률 합성으로 상위/신규 몬스터 획득

---

## 3) 핵심 콘텐츠 3축

### 3.1 섬 경영(생산/확장)
- 자원 생산(농사/벌목) → 건설/업글 재료
- 시설 강화/섬 확장이 장기 목표

### 3.2 전투(방어/디펜스)
- 웨이브 형태로 적이 섬을 공격
- 몬스터 편성/성장이 승패에 직접 영향
- 전투 보상으로 합성 촉매/재료 드랍 가능

### 3.3 몬스터(수집/구매/합성)
- **기본 획득**: A가 퀘스트로 배포한 **특수 재화**로 **몬스터 알** 구매
- **성장 루트**: 중복/하급 몬스터 → **확률 합성**으로 상위/신규 몬스터 획득

---

## 4) 재화/경제 구조

### 4.1 A 포인트(운영 재원)
- 충전: 광고 / 현금 / 출석
- 사용: 퀘스트 보상 재원, 도움 기능 비용 등
- 의미: A의 “지원 예산”

### 4.2 특수 재화(퀘스트 기반, B 성장 키)
- A가 퀘스트 보상으로 배포
- B는 특수 재화로 **몬스터 알 구매**
- 의미: 과금이 아니라 **가족 상호작용(퀘스트)** 중심 성장

### 4.3 섬 자원(플레이 산출물)
- 경영/전투로 획득
- 건설/업글/일부 합성 재료로 사용

### 4.4 합성 관련 재료
- **촉매**: 성공률 보정/특정 계열 합성에 필요
- **정수/조각**: 실패 환급으로 쌓이는 누적 재료(재도전 완충)
- **보호권/안정제**: 실패 손실 방지/성공률 상승 (A 퀘스트 지급에 적합)

---

## 5) 일일 코어 루프(반복 구조)

1. **A가 오늘의 퀘스트 설정**(목표 + 보상 등록)  
2. **B가 섬 플레이(경영/전투)로 퀘스트 수행**  
3. **퀘스트 완료 처리 → 보상 획득**(특수 재화/합성 보조 등)  
4. **몬스터 알 구매 → 몬스터 확보**  
5. **확률 합성으로 상위/신규 몬스터 획득**  
6. **강해진 전력으로 더 높은 방어/더 빠른 성장** → 다음날 반복

---

## 6) A의 방문/도움 시스템(지원 구조)
- A는 B 섬에 자유롭게 들어가 상태 확인 가능
- 도움은 “대신 플레이”가 아니라 **가속(시간 단축/부스트)** 중심으로 설계
- 권장 제한
  - 도움 **횟수 제한(일/주)**
  - 효과량 상한(과도한 단축 방지)
  - 합성 보조(보호권/안정제) 중심 지원으로 B 성취감 보호

---

## 7) 몬스터 확률 합성 시스템(확정 사양)

### 7.1 결과 구조
- 합성 시도 결과는 3가지로 설계 권장
  1) **성공**: 목표 몬스터 획득  
  2) **부분 성공**: 목표보다 한 단계 낮은 결과/대체 보상  
  3) **실패**: 몬스터 미획득, 대신 *환급 + 게이지 누적*

### 7.2 필수 안전장치(가족 게임 핵심)
- **실패 환급**: 정수/조각 형태로 일부 환급
- **천장(피티) 게이지**: 실패/부분성공 시 게이지 증가, 100% 도달 시 **확정 성공**
- **보호권**: 실패 시 재료 1개 보존 등 손실 완화

> 금지 권장: “실패하면 전부 소모, 아무것도 없음” 형태의 완전 날림 실패(갈등/이탈 유발)

### 7.3 운영 원칙(밸런스)
- 성공률 상한 설정(예: 최대 90~95%)
- 보호권 중복 제한(합성 1회당 1개)
- 게이지 즉시 100 만들기 루트는 매우 제한적으로 운영

---

## 8) 전체 시스템 맵(요약)

- A 포인트(광고/현금/출석)  
  → 퀘스트 보상 등록 / 도움 발동  
  → B에게 특수 재화/합성 보조 제공  

- B 플레이(경영/전투)  
  → 자원/드랍 재료 획득  
  → 퀘스트 완료  
  → 특수 재화로 알 구매  
  → 몬스터 편성 강화  
  → 확률 합성(환급+천장+보호)  
  → 전투/섬 성장 가속
