# BaseStats.md

**생존**

* `hp` : 체력
* `hpRegen` : 초당 체력 회복

**공격**

* `ad` : 공격력(물리 기반)
* `ap` : 주문력(마법 기반)
* `as` : 공격속도(초당 공격 횟수 계열로 사용 권장)
* `critChance` : 치명 확률(0~100)
* `critDamage` : 치명 배율(기본 1.5)

**방어/유틸**

* `armor` : 방어력(물리 피해 감소)
* `mr` : 마법저항(마법 피해 감소)
* `moveSpeed` : 이동속도
* `range` : 사거리

**쿨/자원**

* `abilityHaste` : 스킬 가속

**속성**

* `Element` : 속성 추가 데미지
