Firebase 이메일 인증 기반 계정 생성

이메일/비밀번호로 계정을 만들고, 메일로 받은 인증번호(링크의 oobCode)를 입력해야 가입이 완료됩니다.

Firebase 설정이 비어 있으면 먼저 입력 창이 표시됩니다.

가입 절차

  1. 이메일과 비밀번호를 입력한 뒤 인증번호 전송을 누릅니다.
  2. 메일로 받은 인증 링크에서 oobCode 값을 복사하거나 링크 전체를 붙여넣습니다.
  3. 인증번호 확인을 눌러 이메일 인증을 완료하면 계정이 활성화됩니다.

기존에 동일 이메일로 생성된 계정이 있으면 비밀번호 확인 후 인증 메일을 다시 보냅니다.

계정 생성

이 페이지로 돌아온 경우 URL에 포함된 인증번호를 자동으로 인식합니다.
