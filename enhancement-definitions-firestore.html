EnhancementDefinition Firestore 관리

장비 강화 효과(EnhancementDefinition)를 Firestore에서 직접 조회·수정·추가·삭제합니다. 모든 작업 전에 Firebase 설정을 먼저 입력하세요.

강화 정의 입력/편집

저장 시 name을 기준으로 문서 ID를 생성하며, 빈 값·음수 등 안전하지 않은 값은 자동으로 보정됩니다.

기능별 사용 가이드

  • 조회하단 테이블에서 Firestore의 enhancementDefinitions 컬렉션을 즉시 확인합니다.
  • 검색/필터이름·설명 키워드, 부위(slot), 수치 타입(valueType), 사용 여부로 목록을 좁힙니다.
  • 편집행의 "불러오기"를 눌러 양식에 채워진 값을 검토 후 수정·저장합니다. 수정자는 updatedBy 필드에 기록됩니다.
  • 신규 추가필수 필드(name, slot, valueType, baseValue)를 입력하고 저장하면 새 문서를 만듭니다.
  • 삭제사용하지 않는 항목은 삭제 버튼으로 제거합니다. 실행 전 협의 후 진행하세요.

필드 규칙

  • name: 고유 이름. 공백 포함 가능하며 ID는 자동 슬러그화됩니다.
  • slot: weapon/armor/accessory 등 부위 선택.
  • valueType: percent/flat/scaling 중 택1.
  • baseValue · maxValue · step: 0 미만은 0으로 보정됩니다. maxValue가 비어 있으면 무제한.
  • tags: 쉼표 구분 키워드. 공백은 자동 제거합니다.
